# Descripcion del proyecto: taskflow
El proyecto se basa en la creación de un repositorio en github, utilizando el CMD para clonarlo y tenerlo local, añadiendo la creación de una aplicación de consola en C#.

# Como ejecutarlo
Este proyecto contiene un programa simple en C# que imprime "Hello, World!" en la consola. Para ejecutarlo, sigue estos pasos:

### Requisitos previos
* Tener instalado el **SDK de .NET** (puedes descargarlo desde [dotnet.microsoft.com](https://dotnet.microsoft.com/)).
* Una terminal (CMD, PowerShell o la terminal integrada de VS Code).

### Pasos para la ejecución

1. **Abrir la terminal** en la carpeta del proyecto `taskflow`.
2. **Compilar y ejecutar** el programa utilizando el CLI de .NET con el siguiente comando:
   ```bash
   dotnet run
