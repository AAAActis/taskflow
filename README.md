# TaskFlow - Proyecto NovaTech

## Descripción del Proyecto
TaskFlow es una aplicación de consola desarrollada en C# como parte del entrenamiento inicial de NovaTech. Este proyecto establece la base de un sistema de gestión de tareas, enfocándose en la configuración profesional del entorno de trabajo y el control de versiones.

## Tecnologías Usadas
* **Lenguaje:** C# (.NET SDK)
* **Control de Versiones:** Git & GitHub
* **Editor:** Visual Studio Code / Visual Studio
* **Entorno:** Consola (CLI)
